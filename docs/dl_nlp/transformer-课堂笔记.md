今天我们讲Transformer。为什么这个东西重要？因为几乎所有你听说过的AI 模型——Chat-

神经网络基础回顾

✓神经元= 输入× 权重+ 偏置→激活→输出

✓神经网络= 多层神经元堆叠
✓训练= 损失函数+ 梯度下降+ 反向传播

✓文字→分词→编号→Embedding 向量

本节的问题：向量进入网络后，怎么互相“交流”？

先快速回顾一下神经网络基础，确保大家在同一起跑线上：

## • 神经元：就是一个计算单元——输入乘以权重，加个偏置，过一个激活函数，得到输出。数学上

就是y = f(w · x + b)。

• 神经网络：很多神经元堆叠成多层。前一层的输出是后一层的输入。

## • 训练：定义一个损失函数（衡量预测值和真实值的差距），然后用梯度下降不断调整权重，让损

失变小。反向传播负责算出“每个权重该往哪个方向调、调多少”。

• Embedding：文字不能直接喂给模型，需要先分词、编号，再通过Embedding 层变成高维向

量。比如“猫” 这个字可能变成一个4096 维的向量。

这些都搞定之后，现在的核心问题就是：向量进入网络后，怎么互相“交流”？

很简单——同一个向量⃗x，乘以三个不同的矩阵：

⃗q = ⃗x × WQ,
⃗k = ⃗x × WK,
⃗v = ⃗x × WV

其中WQ, WK, WV 是三个可训练的权重矩阵——也就是说模型会自动学习“怎么把输入变成最好

的问题、最好的标签、最好的内容”。

举个例子：假设输入向量⃗x 是4096 维的，WQ 是4096 × 128 的矩阵，那⃗q = ⃗x × WQ 就是128

维的。K 和V 也同理。

为什么要三个不同的矩阵？因为“想找什么” 和“能提供什么” 是不同的事。如果Q 和K 用同一

个矩阵，那每个字只会关注跟自己最像的字——但实际上你想找的和你能提供的往往不一样。分开三

个矩阵让模型更灵活。

⃗a = [1, 2, 3],

点积有什么物理意义？衡量两个向量的“相似度”：

• 两个向量方向相同：点积大→[1, 0] · [1, 0] = 1 →这两个字很相关

• 两个向量方向垂直：点积等于0 →[1, 0] · [0, 1] = 0 →这两个字不相关

• 两个向量方向相反：点积是负数→完全无关

所以：Q 和K 的点积越大→说明当前字和那个字越相关→应该给它更多注意力。

残差连接的核心思想：

output = input + f(input)

也就是说，输出= 原始输入+ 经过处理的结果。原始输入通过“快捷通道” 直接保留。

为什么需要这个？因为LLaMA-7B 有32 层Transformer。如果没有残差连接：

• 原始信息经过32 层计算，可能被“冲淡”

• 梯度经过32 层反向传播，可能“消失”（梯度消失问题）

• 结果就是网络越深越难训练

有了残差连接：

• 原始信息有“快捷通道” 直接跳过某些层

• 梯度可以畅通无阻地回传

• 网络再深也能训练
