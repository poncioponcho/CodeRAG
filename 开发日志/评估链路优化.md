## 二、已解决的问题

### 1. JSON 解析失败
- **现象**：`list indices must be integers` 或 `Extra data` 错误
- **根因**：LLM 返回的 JSON 可能被包裹在 `{"qa_pairs": [...]}` 中，或包含多个片段
- **修复**：在 `generate_test_set_batch.py` 中增加正则提取数组/对象，并自动提取 `qa_pairs` 字段
- **效果**：生成成功率 100%

### 2. 评估覆盖率偏低
- **现象**：相关性 4.0 但覆盖率仅 25%
- **根因**：测试集中包含大量文档无法直接回答的条目（LLM 过度推断）
- **修复**：新增 `filter_test_set.py`，在评估前剔除无法在源文档中找到任何要点的条目
- **预期**：覆盖率提升至 35%~45%
